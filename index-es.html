Simon Dice

Desarrollé un juego de Simon Dice usando LEDs y una protoboard, diseñado e implementado dos veces con diferentes conceptos. El proyecto fue construido para Raspberry Pi, utilizando un servidor backend para vincular una interfaz HTML con lógica en JavaScript. JavaScript manejó la lógica del juego y las interacciones de botones, mientras que el servidor conectó con lógica en C para controlar los LEDs, creando una experiencia interactiva y visualmente atractiva.

Stock ++

Base de Datos de Películas
